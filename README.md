# Catch & Dodge! 🚀

A simple, colorful HTML5 browser game inspired by the kind of quick arcade games you see in mobile advertisements.

## How to play

Open `index.html` in any modern web browser — no installation required.

| Control | Action |
|---------|--------|
| ← / → Arrow keys | Move the spaceship left / right |
| A / D keys | Move the spaceship left / right |
| Touch & drag | Move the spaceship (mobile) |

### Rules

- **Catch** ⭐ stars, 💎 gems, and 🪙 coins to earn points.  
- **Dodge** 💣 bombs or you lose a life.  
- You start with **3 lives** (❤️❤️❤️).  
- The game gets faster and more bombs appear as your score increases (every 15 pts = new level).  
- The game ends when all lives are lost — press **RETRY** to play again.
